# KDP Fantasy Football — Yahoo League IDs by Season

Reference list of the Yahoo Fantasy league ID for each season of the KDP league.
Use these to view any past season directly.

## URL pattern

Past seasons are viewable at:

```
https://football.fantasysports.yahoo.com/<YEAR>/f1/<LEAGUE_ID>
```

Weekly scoreboard / matchups for a given week:

```
https://football.fantasysports.yahoo.com/<YEAR>/f1/<LEAGUE_ID>?matchup_week=<N>&module=matchups&lhst=matchups
```

Standings / schedule:

```
https://football.fantasysports.yahoo.com/<YEAR>/f1/<LEAGUE_ID>?module=standings&lhst=stand
https://football.fantasysports.yahoo.com/<YEAR>/f1/<LEAGUE_ID>?module=standings&lhst=sched
```

Note: the bare `/f1/<id>` form only resolves the *current* season's IDs. For any
past season you must include the `<YEAR>/` prefix as shown above.

## Notes

- 2009 is the WaZZU-era league (different membership; earliest season with reliable
  team–manager records).
- 2010 was previously missing from the project dataset entirely.
- 2026 (107782) is the auto-renewed current shell (pre-draft); its standings display
  2025's final results but its matchup pages are empty until the season is played.
- League IDs are not sequential or chronological — each season is a distinct Yahoo league.
